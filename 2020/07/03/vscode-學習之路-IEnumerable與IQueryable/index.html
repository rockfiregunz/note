vscode-學習之路-IEnumerable與IQueryable

發表於 2020-07-03 分類於 IEnumerable ， IQueryable

使用linq 或是 EF 的時候，常常看文章會使用 IEnumerable、IQueryable
當初看的時候，覺得應該是差不多東西。
但後來還是想要看一下有何差異性，先紀錄一下。

IEnumerable與IQueryable

IQueryable: 處理遠端來源的資料 ex. Database / Service
等到要使用資料才會轉成語法，，取出符合條件的

IEnumerable: 處理記憶體內的資料 ex. List / Array
先取出資料，在比較符合條件，因此資料量多的時候，可能會有問題產生
資料量有10萬筆，先將10萬筆找出來丟到記憶體，再去拿取需要的5筆資料
