1-hexo建立

發表於 2020-02-07 更新於 2020-02-13 分類於 Hexo

只是單純地想記錄一下使用 hexo 的過程。

版本

  • windows 　10
  • hexo　　　4.2.0
  • git　　　　2.19
  • node　　　v10.13.0

下載及安裝

請到 git官網 及 hexo 官網 下載安裝

安裝套件 hexo-cli

$ npm install hexo-cli -g

安裝套件 Hexo Git

$ npm install hexo-deployer-git --save

初始 hexo

$ hexo init blog     # blog 自訂的名稱
$ cd blog           
$ npm install        # 安裝相關套件

啟動 hexo

$ hexo s             # hexo server 縮寫

安裝 hexo 主題 NEXT

git clone https://github.com/theme-next/hexo-theme-next themes/next
  • 我是降版本使用 v7.7.0 版本，因為 v7.7.1 我安裝之後都有問題

發佈到github

到github 申請一個帳號，建立一個 Repositories
Repositories 名稱 : 帳號.github.io

設定 config.yml 的檔案，這是 Hexo 的全域配置文件

  1. 到檔案最下面，更改設定

    repository 這個可以用 https的方法，請上網查看。
    我自己是用 ssh ，所以要設定 ssh key來使用。

清除 及 產生檔案

$ hexo clean           # 清除檔案
$ hexo g               # hexo generate 縮寫，產生檔案

上傳到github

$ hexo d              # heox deploy  縮寫
